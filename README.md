🌤️Weather Application
A simple weather app built as part of the SheCodes program. Users can search for any city and see the
currentweather conditions along with a short forecast.


🔗 Live demo: https://weatherscope-web.netlify.app/


✨Features:
-Search current weather by city name
-Get weather for your current location using geolocation
-View temperature, humidity, wind speed, and general conditions
-Multi-day weather forecast
-Toggle between Celsius and Fahrenheit
-Responsive design for mobile and desktop

🛠️Built With:
HTML5
CSS3
JavaScript
OpenWeatherMap API

Prerequisites:
A free API key from OpenWeatherMap

🎯What I Learned
This project helped me practice making API calls, handling asynchronous JavaScript, manipulating the DOM 
based on API responses, formatting dates, and building a responsive layout with plain CSS.
